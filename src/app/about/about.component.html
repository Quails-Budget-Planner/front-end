About the Project

Jonathan and Jordan are fully endorsed graduates from Lambda School's Full-Stack Web Development track.

They've collaborated on several applications together, including:

  • QualityHub + InterviewQ
  • Create Your Own Adventure
  • The Legend of Lambda

For this project, they wanted to implement an unfamiliar web development technology stack and utilize some AWS services for database management and application deployment.

This application is built with Angular, Express, and Amazon DynamoDB. It's styled with Angular Material.
